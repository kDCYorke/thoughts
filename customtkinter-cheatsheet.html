Setup & App Skeleton
import customtkinter as ctk

# Appearance: "System" | "Dark" | "Light"
ctk.set_appearance_mode("Dark")
# Theme: "blue" | "green" | "dark-blue"
ctk.set_default_color_theme("blue")

class App(ctk.CTk):
    def __init__(self):
        super().__init__()
        self.title("My App")
        self.geometry("800x500")
        self.minsize(400, 300)
        self._build_ui()

    def _build_ui(self):
        self.grid_columnconfigure(0, weight=1)
        self.grid_rowconfigure(0, weight=1)

if __name__ == "__main__":
    app = App()
    app.mainloop()

Widget Overview
CTkButton
Clickable button with hover, rounded corners, optional image
CTkLabel
Static or dynamic text label, supports wrapping & images
CTkEntry
Single-line text input, placeholder text, show="*" for passwords
CTkTextbox
Scrollable multi-line text editor (replaces tk.Text)
CTkCheckBox
Toggle with BooleanVar; onvalue / offvalue supported
CTkRadioButton
Grouped exclusions via shared IntVar / StringVar
CTkSwitch
iOS-style toggle switch with on/off callbacks
CTkSlider
Horizontal/vertical range slider, from_/to, number_of_steps
CTkProgressBar
Determinate or indeterminate progress indicator
CTkComboBox
Dropdown + editable field; values=[] list, command callback
CTkOptionMenu
Dropdown only (no free text); dynamic_resizing option
CTkSegmentedButton
Tab-like multi-option selector; values=[]; command callback
CTkTabview
Tabbed pane; .add("Name") returns a frame per tab
CTkScrollableFrame
Auto-scrollbar container; orientation="vertical"/"horizontal"
CTkFrame
Container/panel; fg_color, corner_radius, border_width
CTkImage
HiDPI-aware image; light_image, dark_image from PIL

Tkinter Variables
# Reactive variables (use with widgets)
sv = ctk.StringVar(value="hello")
iv = ctk.IntVar(value=0)
bv = ctk.BooleanVar(value=False)
dv = ctk.DoubleVar(value=3.14)

sv.set("world")
print(sv.get())

# Trace changes
def on_change(*args):
    print("changed:", sv.get())
sv.trace_add("write", on_change)

class SettingsWindow(ctk.CTkToplevel):
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self.title("Settings")
        self.geometry("400x300")
        self.grab_set()  # modal behaviour

# Open from main window
def open_settings(self):
    win = SettingsWindow(self)
    win.focus()

Tips, Gotchas & Patterns
Topic Advice
Mixing tk + ctk Avoid mixing tk.Widget with CTk containers — appearance won't match; prefer CTk equivalents throughout.
fg_color="transparent" Use on CTkFrame to let the parent background show through — useful for nested panels.
Thread safety Never update widgets from a background thread. Use self.after(0, callback) to queue updates on the main loop.
Image refs Store CTkImage as an instance attribute (e.g. self.img = …) to prevent garbage collection causing blank images.
Responsive layout Always pair grid(sticky="nsew") with grid_columnconfigure(col, weight=1) on the parent.
CTkFont f = ctk.CTkFont(family="Helvetica", size=14, weight="bold") — pass to font= param for consistent scaling.
Custom theme Pass a JSON file path to set_default_color_theme("my_theme.json"). Copy an existing theme from the CTk source as a starting point.
DPI on Windows Call ctk.deactivate_automatic_dpi_awareness() before creating the app if you manage DPI yourself.
